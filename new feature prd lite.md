# PRD: 실험실 — 경제적 주유소 찾기

## 개요

단순 가격 순위가 아닌, **내 차의 연비와 실제 이동 거리를 반영한 순 주유량 기준**으로 가장 경제적인 주유소를 찾아주는 기능.  
홈 탭 하위의 **실험실(Lab)** 탭에 독립 배치한다.

---

## 배경 및 문제 정의

기존 서비스는 주유소별 리터당 가격만 표시한다.  
그러나 가장 싼 주유소가 멀리 있을 경우, 그곳까지 가는 데 소모되는 연료 비용이 가격 이점을 상쇄할 수 있다.  
사용자에게 필요한 것은 "가장 싼 주유소"가 아니라 **"내 상황에서 가장 경제적인 주유소"** 다.

---

## 사용자 입력값

| 항목 | 설명 | 예시 |
|---|---|---|
| 출발지 | 내 위치 사용 **또는** 텍스트 주소 직접 입력 | "광주 북구 용봉동" |
| 연비 | 사용자 차량의 평균 연비 (km/L) | 11.5 |
| 목표 주유 금액 | 이번에 주유할 금액 (원) | 60,000 |

> 연비와 목표 주유 금액은 **로컬스토리지에 저장**하여 재방문 시 자동 입력.

### 출발지 입력 상세

두 가지 방식을 UI에서 선택 가능하도록 구성한다.

**방식 A — 내 위치 사용 (기본값)**
- 브라우저 Geolocation API로 현재 좌표 획득
- 위치 권한 허용 시: 좌표를 카카오 좌표→주소 API로 역지오코딩해서 주소 표시
- 위치 권한 거부 시: 방식 B로 자동 전환하며 안내 메시지 표시

**방식 B — 주소 직접 입력**
- 텍스트 입력 + 카카오맵 주소 검색 API로 좌표 변환

---

## 핵심 계산 로직

```
총 주유량(L) = 목표 주유 금액 ÷ 리터당 가격
이동 연료 소모량(L) = 왕복 거리(km) ÷ 연비(km/L)
순 주유량(L) = 총 주유량 - 이동 연료 소모량
```

> **왕복 거리** = 카카오 모빌리티 길찾기 API로 출발지 → 주유소 편도 거리 × 2  
> 단, 주유 후 귀환 경로가 다를 수 있으므로 편도 거리만 사용하는 옵션도 고려 가능 (v2 검토)

## 데이터 소스

### 주유소 가격 데이터 — GitHub Actions 캐시 (API 호출 없음)
매일 한국 시간 06:00, 18:00에 GitHub Actions가 전체 주유소 가격을 수집해 캐시 파일로 저장한다.  
실험실 기능은 이 최신 캐시 파일을 그대로 읽어 사용한다. 외부 가격 API를 별도 호출하지 않는다.

### 거리 계산 — 카카오 모빌리티 길찾기 API (호출 최소화)
모든 주유소를 대상으로 거리 API를 호출하면 무료 한도를 초과할 수 있다.  
따라서 캐시 파일에서 **가격 기준 상위 N개(기본값: 10개)** 를 먼저 필터링한 뒤,  
해당 주유소에 대해서만 길찾기 API를 호출한다.

> N값은 API 사용량 모니터링 후 조정 가능하도록 상수로 분리한다.

## 외부 API

| API | 용도 |
|---|---|
| 브라우저 Geolocation API | 현재 위치 좌표 획득 (방식 A) |
| 카카오맵 좌표→주소 API | 현재 좌표 → 주소 변환 (방식 A, 역지오코딩) |
| 카카오맵 주소 검색 API | 텍스트 주소 → 좌표 변환 (방식 B) |
| 카카오 모빌리티 길찾기 API | 출발지 ↔ 주유소 간 실제 경로 거리 조회 |

## 결과 화면 설계

### 정렬 기준
순 주유량(L) 내림차순 — 가장 많이 넣을 수 있는 주유소가 1위.

### 결과 카드 표시 항목
- 주유소명
- 리터당 가격
- 출발지까지 경로 거리 (km)
- 총 주유량 (L)
- 이동 연료 소모량 (L)
- **순 주유량 (L)** ← 핵심 지표, 강조 표시

### 뱃지
- 1위 카드에 `✅ 가장 경제적` 뱃지 표시
- 가격 1위 주유소가 순 주유량 1위가 아닐 경우, 가격 1위 카드에 `💰 가장 저렴` 뱃지 별도 표시

### 프레이밍 원칙
금액 절약 강조 X → **"내 상황에서 최선의 선택"** 확인 중심으로 카피 작성.  
예: `OO주유소에서 60,000원으로 가장 많이 주유할 수 있어요 (순 28.86L)`

## UX 플로우

```
실험실 탭 진입
→ 출발지: [내 위치 사용] 버튼 또는 주소 직접 입력
   - 내 위치 사용: 권한 허용 시 좌표 획득 + 주소 표시 / 거부 시 입력창으로 전환
→ 연비 입력 (이전 값 자동 채워짐)
→ 목표 주유 금액 입력 (이전 값 자동 채워짐)
→ [경제적 주유소 찾기] 버튼
→ 캐시 파일에서 가격 상위 N개 필터링
→ 로딩 (필터링된 N개에 대해서만 길찾기 API 호출)
→ 결과 리스트 (순 주유량 기준 정렬)
```

## v2 로드맵

### 거리 반경 필터링
유저가 검색 반경(km)을 직접 설정하면, 해당 범위 내 주유소만 대상으로 계산한다.

구현 흐름:
1. MongoDB `$geoNear` 쿼리로 반경 내 주유소 좌표 조회 (이미 2dsphere 인덱스 적용됨)
2. 캐시 파일의 최신 유가와 주유소 ID 기준으로 조인
3. 조인된 후보에 대해서만 카카오 길찾기 API 호출

### 경유 지원
v1 휘발유 전용에서 경유까지 확장. 유종 선택 UI 추가.
